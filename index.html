At Behind The Window Team, we develop Android applications with a strong commitment to user privacy. We believe your data belongs to you, and we're transparent about how we handle it.

Core Principles:

  • No Unnecessary Data Collection: We only collect data essential for app functionality
  • Local First: Your data stays on your device whenever possible
  • Transparency: Clear disclosure of all data practices
  • Security: Implementation of industry-standard security measures
  • User Control: You remain in control of your data

üìä Information We Don't Collect

We do not collect the following personal information:

  • Personal identification (name, email, address)
  • Contact lists or address books
  • Location data (unless explicitly required by app functionality)
  • Photos, videos, or media files
  • Payment or financial information
  • Sensitive device information

What We May Access:

Some apps may request permissions for specific features:

  • Storage Access: For saving/loading files locally
  • Internet Permission: For app updates or cloud features
  • Camera Access: Only for apps with camera functionality

Each permission is clearly explained during installation and can be revoked at any time.

üîó Third-Party Services

1. Google Play Services

Our apps are distributed through the Google Play Store. Google may collect:

  • Installation and uninstallation statistics
  • Anonymous crash reports (when you opt-in)
  • Aggregate usage metrics
  • Device information for compatibility purposes

This data collection is governed by Google's Privacy Policy. We have no control over Google's data practices.

2. Firebase Services (If Used)

Some apps may use Google Firebase for:

  • Anonymous analytics to improve app performance
  • Cloud messaging for notifications (with user consent)
  • Remote configuration for feature management

All Firebase data is anonymized and governed by Firebase Privacy Policies.

3. Advertising Services

If an app contains ads, we use:

  • Non-personalized ads by default
  • GDPR and CCPA compliant ad networks
  • Transparent ad labeling

üíæ Data Storage & Security

Local Storage

Most app data is stored locally on your device in:

  • App-specific internal storage (sandboxed)
  • Encrypted SharedPreferences for sensitive data
  • SQLite databases with proper access controls

Cloud Storage (If Applicable)

For apps with cloud backup features:

  • Data is encrypted in transit (HTTPS/TLS)
  • Encryption at rest on secure servers
  • User authentication required for access
  • Data deletion upon account removal

Security Measures

  • Regular security audits and updates
  • Code obfuscation and anti-tampering measures
  • Secure API communication
  • Timely security patch implementation

üë∂ Children's Privacy

Our applications are designed to be family-friendly:

  • We comply with COPPA (Children's Online Privacy Protection Act)
  • No personal information collected from children under 13
  • Parental consent mechanisms where required
  • Age-appropriate content and advertising

If you believe we have inadvertently collected information from a child under 13, please contact us immediately for deletion.
